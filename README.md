# 🌀 PYbrowser

A lightweight web browser built using **Python** and **PyQt5**, designed to demonstrate desktop application development and core browser functionality.

---

## 📌 Overview

**PYbrowser** is a graphical web browser application developed with PyQt5. It provides essential browsing features within a simple and minimal user interface.

This project is useful for:
- Learning GUI development with Qt  
- Understanding basic browser workflows  
- Exploring Python-based desktop applications  

---

## ✨ Features

- Tabbed browsing  
- URL navigation and search bar  
- Back, forward, and reload controls  
- Video streaming support  
- Bookmark management  
- History tracking  
- Clean and intuitive UI  

---

## 🛠️ Tech Stack

- **Python (3.7+)**  
- **PyQt5**  
- **Qt WebEngine**
